РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ 

Настоящая программа предназначена для пользователей и сотрудников библиотеки университета.

Главное окно представляет собой две кнопки - это варианты использования система. Как читатель
и как библиотекарь. Чтобы войти в систему как библиотекарь необходимо знать пароль.

ОПИСАНИЕ ОКНА ПОЛЬЗОВАТЕЛЬ

Пользователю доступны данные о книжной базе бибилиотеке. Есть возможность совершить три вида
запросов:
	- По автору книги;
	- По названию книги;
	- По жану.
Снизу располагается кнопка которая возвращает в главное окно.

ОПИСАНИЕ ОКНА БИБЛИОТЕКАРЬ

Возможности:
	- Добавить читателя;
	- Удалить читателя;
	- Изменить данные читателя;
	- Выдать книгу читателю;
	- Показать список всех заказов. В свою очередь в окне с заказами можно сделать запрос по Читателю, Книге, Роду деятельности и статус заказа.
	- Добавить поступившую партию книг. А так же совершить поиск по книгам в данном разделе.
